# 湯気サウナLP - README

## 📁 プロジェクト構成

```
yuge-sauna-lp/
├── index.html          # メインページ
├── style.css           # スタイルシート
├── script.js           # JavaScript
├── images/             # 画像フォルダ
│   └── README.txt     # 画像エクスポート手順
└── README.md          # このファイル
```

## 🚀 使い方

### 1. ファイルの確認
すべてのファイルがダウンロードされていることを確認してください。

### 2. 画像の追加
`images/` フォルダ内の `README.txt` を参照し、Figmaから画像をエクスポートしてください。

### 3. ブラウザで開く
`index.html` をダブルクリックして、ブラウザで開きます。
