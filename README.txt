--------------------------------------------------------------------
Figmaからのエクスポート手順
--------------------------------------------------------------------

【方法1: 個別にエクスポート】

1. Figmaでデザインファイルを開く

2. エクスポートしたい画像を選択
   - 背景画像の場合: フレーム全体を選択
   - 写真の場合: 画像要素を選択

3. 右パネルの「Export」セクションを確認
   - ない場合: 右下の「+」ボタンをクリック

4. 形式を選択
   - 写真: JPG（推奨）
   - アイコン: SVG または PNG

5. 「Export」ボタンをクリック

6. ダウンロードした画像を、このフォルダ（images/）に移動

7. ファイル名を上記のリストに合わせて変更


【方法2: まとめてエクスポート】

1. Figmaで左側のレイヤーパネルを開く

2. エクスポートしたい画像を複数選択
   - Cmd/Ctrl を押しながらクリック

3. 右クリック → 「Export」

4. 形式を選択してエクスポート

5. ダウンロードしたファイルをこのフォルダに移動

6. ファイル名を変更

--------------------------------------------------------------------
画像がない場合
--------------------------------------------------------------------

画像がなくても、HTMLは表示されます。
ただし、以下のようになります：

- ファーストビュー: グレーの背景
- 各セクション: 画像なしでテキストのみ表示

まずは画像なしで動作確認し、後から追加することも可能です。

--------------------------------------------------------------------
推奨される画像の最適化
--------------------------------------------------------------------

エクスポート後、以下のツールで画像を圧縮することをお勧めします：

- TinyPNG (https://tinypng.com/)
- Squoosh (https://squoosh.app/)
- ImageOptim (Mac用アプリ)

圧縮することで、ページの読み込み速度が向上します。

--------------------------------------------------------------------
確認方法
--------------------------------------------------------------------

画像を追加したら：

1. index.html をブラウザで開く
2. 各セクションで画像が表示されているか確認
3. 表示されない場合:
   - ファイル名が正しいか確認
   - ファイルがこのフォルダ（images/）内にあるか確認
